How to Play

The Basics

This is a 3D version of the classic Tick-Tack-Toe game, there are three boards of 3x3 cells each. To win, a player must must have a series of 3 consecutive cells marked with the same symbol either within the same board or across the 3 boards.

Some Examples of Wins

3 across on a single board
3 across on all boards
Diagonal on all boards
The same cell on all boards
